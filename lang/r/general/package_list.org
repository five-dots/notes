* Workflow

_R のワークフローパッケージ_
- R Package の構造を利用して、プロジェクトのディレクトリ構造を自動作成するもの
- データの管理 (キャッシュ・自動読み込み) 機能をもつもの
- RMarkdonw/knitr を利用してレポーティングを自動化するもの
- git を使って、コードだけでなく、リサーチ結果もバージョン管理するもの
